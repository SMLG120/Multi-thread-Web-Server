⚡ COMP2322 Web Server

Multi-Threaded HTTP/1.1 Server — Python Socket Implementation

● SERVER ONLINE

📡 Supported Features

  • ✅ Multi-threaded (one thread per client)
  • ✅ GET — text & image files
  • ✅ HEAD — headers only
  • ✅ Persistent connection (keep-alive)
  • ✅ Non-persistent connection (close)
  • ✅ Last-Modified / If-Modified-Since
  • ✅ Request logging to file

🔢 Response Status Codes

  • 200 — OK
  • 304 — Not Modified
  • 400 — Bad Request
  • 403 — Forbidden
  • 404 — Not Found
